
# 🐍 PyMentor AI

PyMentor AI is an AI-powered Python learning assistant built with Streamlit and Google's Gemini AI.

It helps beginners learn Python interactively through AI-generated explanations, code generation, debugging assistance, quizzes, revision notes, and interview preparation.

---

## 🚀 Features

- 📖 Learn Python Concepts
- 💻 Python Code Generator
- 🐞 Debug Python Code
- 📝 AI Quiz Generator
- 📚 Revision Notes
- 🎤 Python Interview Questions

---

## 🛠 Technologies Used

- Python
- Streamlit
- Google Gemini API
- HTML & CSS
